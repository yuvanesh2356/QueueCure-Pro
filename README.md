# QueueCure Pro

QueueCure Pro is a smart healthcare queue management system built using React, Vite, Firebase Authentication, and Cloud Firestore.

## Features
- Real-time queue tracking
- QR-based patient check-in
- Receptionist dashboard
- Live token management
- TV display for waiting rooms
- Estimated waiting times

## Tech Stack
- React
- Vite
- Firebase Authentication
- Cloud Firestore
- Tailwind CSS
